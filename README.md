# 柳比歇夫時間管理 App
Demo: https://time.kaului.com/

## Tech Stacks

### Front-end
Vite + TypeScript + React

### Back-end
Ruby on Rails API + PostgresSQL + Active Record ORM

### Deployment
Dokku (Heroku liked service)

### Auth
Telegram Login Widget
